Sobre os modelos:
• ECMWF IFS — modelo determinístico europeu, considerado o mais preciso do mundo para previsão de médio prazo (7–15 dias), especialmente para eventos de precipitação.
• GFS (NOAA) — Global Forecast System americano, excelente para regiões tropicais, disponível gratuitamente a cada 6h com alcance de 16 dias.
• Ambos acessados via Open-Meteo API (gratuita, sem chave).
